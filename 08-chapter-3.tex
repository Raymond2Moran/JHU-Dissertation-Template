\chapter{Protein Edge Prediction on Intra Splits} \label{chap:protein_main}

\section{Experimental Setup}

Models are trained on Intra1, tuned on Intra0, and evaluated on Intra2. The evaluation compares sequence-only, text-only, and sequence+text features when available. For each setting, we export canonical prediction tables and run budgeted PPI/PPI++ inference at 5\%, 10\%, and 20\% labeled fractions.

\section{Predictive Performance}

Table files generated by the pipeline summarize point metrics (accuracy, F1, ROC AUC, PR AUC), while bootstrap outputs quantify AUC uncertainty. The full tables are exported in the thesis workspace under \texttt{results/tables/}.

\section{Interim Conclusions}

Across Intra splits, representation quality and calibration-budget size jointly determine final inferential precision. Sequence model choice influences both predictive performance and downstream efficiency in prediction-powered estimators.
