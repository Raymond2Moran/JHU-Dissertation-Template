\chapter{Text Features and External Robustness Analysis} \label{chap:robustness}

\section{Text-Feature Augmentation}

This chapter evaluates whether adding UniProt description embeddings improves edge prediction and inferential stability relative to sequence-only models. Pair-level text vectors are combined with sequence vectors in a concatenated feature representation.

\section{External Dataset Construction}

A BIOGRID-human physical-interaction dataset is built with UniProt-mapped proteins, matched negative sampling, and protein-disjoint splits to reduce leakage through shared nodes. This external test assesses transportability beyond the original benchmark partitions.

\section{Robustness Interpretation}

The external analysis highlights two practical points:
\begin{enumerate}
    \item Improvements in point prediction do not always translate into equal uncertainty reduction under limited labels.
    \item Feature fusion can provide gains, but robustness should be judged with confidence intervals and not only point metrics.
\end{enumerate}

\section{Chapter Summary}

Text-augmented models provide an important extension to sequence-only pipelines, and external-set evaluation confirms the need for leakage-aware design and uncertainty-aware reporting.
